Visualizing lakeshore data



The interpolation routine I wrote here uses scipy-spline interpolation


Show below is a zoomed in section of the lakeshore data, its difficult to visualize the interpolation with the density of datapoints.

Range of the x-axis can be modified according to need, for sample I am showing data between 1V and 1.2V



Common method of testing models is to separate dataset into sample and test data and see how the model trained with sample data performs for the test data


Following plot shows the error from comparing interpolated points on odd-datapoints (as test data) using even_datapoints (as sample data)

Error is simply calculated as the difference of temperatures between odd-datapoints and interpolated points


Maximum error observed (in K) 0.15020296614636663 At V = 0.621141

Average error observed (in K) 0.005930609463249136


With the error obtained by interpolating over the odd-data points, we obtained a maximum difference of ~0.15K

This dictates that the max error that the interpolation over all data would be under ~0.15K due to the smaller step size
